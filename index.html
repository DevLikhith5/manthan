What it does

Semantic search

Finds code by intent, not keywords. Understands what you mean without exact names.

Local & private

Everything runs locally. Code never leaves your machine. No SaaS, no data leaks.

Multi-repo

Index and search across multiple repos. Monorepos, microservices — all work.

Citation-backed answers

Every answer includes exact file paths and line numbers. Click to open source.

How it works
1

Clone & index

Add a repo. Manthan clones it locally and builds a hybrid search index (embeddings + BM25).

2

Ask in English

Type questions like "how does auth work". No regex, no query language — just plain English.

3

Get answers with sources

Synthesized answer with exact file paths and line numbers. Click any citation to open.
